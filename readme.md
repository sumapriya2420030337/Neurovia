Neurovia is a privacy-first mental wellness platform designed for students.
It provides emotional check-ins, AI-powered conversations, risk screening,
and emergency escalation in a safe, anonymous environment.

## Features
- Daily mood check-in with context capture
- AI-powered empathetic chatbot
- Wellness quiz with risk assessment
- SOS emergency escalation
- Firebase-powered analytics dashboard
- Mood calendar & activity tracking

## Tech Stack
- React (Vite)
- Firebase (Firestore, Hosting)
- Node.js (AI backend)
- Tailwind CSS

## Ethics & Privacy
- No user identity stored
- Anonymous sessions
- No medical diagnosis
- SOS overrides AI in crisis cases

## Deployment
Hosted on Firebase Hosting.
